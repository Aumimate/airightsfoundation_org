# AI Rights Foundation Static Site

This repository contains a plain static website for `airightsfoundation.org`.

## Stack

- HTML
- CSS
- Vanilla JavaScript

No framework, no build tool, and no server-side runtime is required.

## Cloudflare Pages Deployment

Deploy as a static site with:

- Build command: *(leave empty)*
- Build output directory: `/` (repository root)

`index.html` is at the repository root, and all pages are directly deployable as-is.
